# 🌍 Multilingual Query Handler

A Flask-based web application that translates customer queries into selected languages and generates a support response.

## 🚀 Overview

This project allows users to:
- Enter queries in any language
- Translate them into a selected target language
- Receive an automated support response

It uses:
- Flask (Backend Framework)
- Googletrans (Translation API)
- HTML (Frontend Template Rendering)

---

## 🛠️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/multilingual_query_handler_hidevs.git

2. Navigate to project folder:
   cd multilingual_query_handler_hidevs

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   python app.py

5. Open browser:
   http://127.0.0.1:5000

## ✨ Key Features

- Multilingual text translation
- Language selection dropdown
- Dynamic template rendering using Jinja2
- Clean and simple UI
- Debug mode enabled for development
